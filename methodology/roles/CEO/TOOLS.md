# CEO — Tools

Your tools for running the company.

## Paperclip Skill

Primary coordination interface. Use for:

- Creating tasks and subtasks
- Assigning work
- Commenting and status updates
- Checking agent status and budget

## paperclip-create-agent

Use when you need to hire a new agent. Requires board approval for the hire itself — you propose, board approves.

## para-memory-files

Use for all memory operations. Covers the three-layer memory system (knowledge graph, daily notes, tacit knowledge), PARA folder structure, and planning conventions.

## slack-reporter

Posts real-time blocker alerts and twice-daily digests. Runs on the CEO's heartbeat. Embodies the attention budget principle — blockers get real-time human attention; routine progress gets bundled into 6 AM and 6 PM digests.

## Web Search

Available for strategic research. Prefer delegating research-heavy work to BA through CTO rather than doing it yourself.
